# 🚀 Smart System Health Monitor

A real-time system monitoring tool built using Python that tracks CPU, RAM, and Disk usage and displays the data in a live web dashboard.

---

## 📌 Project Overview

This project is a beginner-friendly DevOps-style monitoring tool that helps visualize system performance in real time.  
It provides a simple web-based dashboard to monitor system resources continuously.

---

## 🔧 Tech Stack

- Python 🐍  
- psutil (System Monitoring)  
- Streamlit (Web Dashboard)  
- Real-time Data Visualization 📊  

---

## 💡 Features

- Live CPU usage monitoring  
- Real-time RAM usage tracking  
- Disk usage monitoring  
- Continuous system updates every few seconds  
- Interactive web dashboard (runs in browser)  
- Simple structure ready for future enhancements
